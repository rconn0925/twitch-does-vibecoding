## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-19 | Elevation of Privilege (stored XSS, broadcast) | overlay.js rendering chat-derived candidate titles | mitigate | textContent-only el() helper; 80/60-char JS truncation + CSS ellipsis; no URL/image/markup interpretation ever; machine-checked by tests/invariants/dom-safety.test.ts scanning ALL src/**/public/*.js for innerHTML/outerHTML/insertAdjacentHTML/document.write/eval |
| T-02-20 | Elevation of Privilege | overlay surface exposing operator controls (D2-17) | mitigate | Physically separate http server + port from the console; zero mutation routes by construction (grep-gated: no app.post/put/delete/patch); 127.0.0.1 bind; ws Origin check copied from console (WR-01) |
| T-02-21 | Information Disclosure | overlay leaking internal state on stream (halt alarm, errors, rationales) | mitigate | Pill vocabulary is a fixed public mapping (ON HOLD, never "HALTED"); no error text ever renders on the broadcast surface (freeze-and-retry contract); overlay state JSON carries only round snapshot + queue titles — no audit/review/rationale data in buildOverlayState() |
| T-02-22 | Denial of Service | per-vote push flooding OBS CEF rendering (flicker on stream) | mitigate | 300ms trailing debounce on VOTE_RECORDED in overlay/server.ts; CSS width transition smooths debounced jumps; immediate pushes reserved for low-frequency lifecycle events |
