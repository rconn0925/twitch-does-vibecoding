Fix the stale app-under-construction preview (confirmed live 2026-07-16: OBS showed app-5 while 5555 verifiably served app-6). Two stacked defects, two surgical fixes:

Purpose: the OBS LIVE BUILD slot must always show what 5555 actually serves — mid-build (files changing under the same URL) and across project switches (new generation behind the same URL).
