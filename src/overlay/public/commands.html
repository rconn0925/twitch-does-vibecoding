VOTE COMMANDS

  • !build <idea> start a brand-new project
  • !suggest <tweak> change or add to what's on screen
  • !swapbuild "name" switch to another project in the portfolio
  • !revert undo the last change (alias !undo)
  • !vote 1-5 pick an option in the live round
  • !chaos vote to trigger a random-pick chaos window

INFO COMMANDS

  • !projects list the current portfolio
  • !current show what's building right now
  • !repo get the project's repo link
  • !help show this command list
